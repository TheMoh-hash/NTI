# 🏎️ NTI - Location de Véhicules de Luxe

**NTI** est une vitrine web élégante et responsive pour une agence de location de voitures de prestige. Ce projet met en avant la présentation haut de gamme des véhicules et une navigation intuitive pour les utilisateurs.

---

## 🎨 Conception et Design
Le design a été pensé pour offrir une immersion immédiate dans l'univers du luxe. L'utilisation du framework **Bootstrap** a permis de garantir :
- **Adaptabilité :** Une interface fluide sur tous les supports (Desktop, Tablettes, Mobiles).
- **Rigueur visuelle :** Une grille cohérente et des composants UI professionnels.
- **Réactivité :** Une navigation optimisée pour convertir les visiteurs en prospects.

## 🛠 Stack Technique
- **Structure :** HTML5 (Code sémantique pour un meilleur SEO).
- **Style & Mise en page :** Bootstrap 5, complété par du CSS3 personnalisé pour les détails de finition (animations, typographie).
- **Gestion des ressources :** Organisation optimisée des assets (images haute définition, icônes).

## 🚀 Installation locale
1. Cloner le dépôt : 
   `git clone https://github.com/TheMoh-hash/NTI.git`
2. Ouvrir le dossier :
   `cd NTI`
3. Ouvrir le fichier `index.html` dans votre navigateur.
